## Editing
To change which sites are shown, edit [sites.json](sites.json).

**Note**: All sites must have a `"subsites": [ ... ]` tag. If any site is missing this tag then it (and everything after it) will not be loaded. Subsites must also have that tag; however, the list must be empty or else things will break (namely, clicking on the `Â»` won't work correctly for subsites).
